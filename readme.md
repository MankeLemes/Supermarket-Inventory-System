# 📦 Inventory Control System

## 📌 Description
This project simulates a supermarket stock management system developed in Python. It allows users to register products, manage inventory, track stock movements, and generate management reports.

## ⚙️ Features
- Product registration
- Product search
- Product listing
- Product update
- Product deletion
- Stock entry registration
- Stock exit registration (sales)
- Movement history tracking
- Reports menu
- Most expensive product
- Cheapest product
- Highest stock product
- Lowest stock product
- Total stock value
- Critical stock products

## 🚀 How to run
1. Open the terminal
2. Run:
   ```bash
   python main.py
